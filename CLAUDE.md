# CLAUDE.md

本目录是标准文件库 `标准文件/`，遵循 **第一性原理 + 丰田 5S + ISO 9001** 维护。

## 第一性原理

```
标准索引的本质 = 在正确的时间，以最小的成本，找到正确的现行标准。

推演出的设计决策:
  主目录只存现行标准（过期/撤销另册归档至 00_过期版本/）
  一文件一行索引（四维定位：路径/国际等效/产品线/试验类型）
  异常高亮（缺增补/待核实 黄色，缺文件 红色）
  PDCA 闭环维护（年审计 + 季抽查）
```

## 命名规范

```
[组织]_[标准号]-[年份]_[名称]_[语言].扩展名

组织前缀:  API_ / ASTM_ / ISO_ / GBT_ / SYT_ / GOST_ / NACE_ / FM_
语言后缀:  _CN (中文) / _EN (英文) / _RU (俄文)，中文默认可省略
分隔符:    统一使用下划线 `_`，禁止空格和短横混用
旧版本:    移至 00_过期版本/，原文件名不变

示例:
  API_15HR-2016_高压玻璃钢管_CN.pdf
  GBT_29165.1-2022_石油天然气工业_玻璃纤维增强塑料管_第1部分-词汇、符号、应用及材料.pdf
  ISO_14692-1-2017.pdf
```

## 工作约定

### 新增标准
1. 按命名规范重命名文件
2. 归入对应 `0X_组织/` 目录（按组织字母简称）
3. 在 Excel 01-Sheet 新增一行
4. 更新 05-Sheet 统计

### 标准过期
1. 核实官方最新版信息
2. 将过期文件移至 `00_过期版本/`
3. 在 Excel 中标记为归档，从 01-Sheet 移除
4. 在 06-Sheet 新增归档记录

### 标准查新（按 PDCA）
- **P (年)**: 每年 12 月对照 ASTM/API/ISO/GB 官网批量查新
- **D (即时)**: 新增标准即时录入
- **C (季)**: 抽查文件位置、版本、命名
- **A (即时)**: 发现过期→归档 / 缺增补→标记 / 新发布→采购

### 禁止事项
- 活跃目录存过期标准（先归档再加新）
- 文件名含空格、短横、来源标记（如 `_免费标准网`）
- `.DS_Store` 残留
- 三级及以上子目录
- Excel 索引与文件系统不一致

## 常见任务

### 查找标准
- 按标准号找 → Excel 01-Sheet 搜索标准编号
- 按产品线找 → Excel 02-Sheet 按产品类别浏览
- 按检测项目找 → Excel 03-Sheet 按试验方法浏览
- 找国际替代 → Excel 04-Sheet 等效矩阵

### 版本核查
- 查 API 最新版 → apiwebstore.org 搜索标准号
- 查 ASTM 最新版 → astm.org/standards 搜索
- 查 ISO 最新版 → iso.org 搜索
- 查 GB/T 最新版 → std.samr.gov.cn 搜索
